## 為什麼成為數據高速公路上的硬件挖礦者？

若硬件礦工承諾在一段時間內保持物聯網硬件的運行，將得到獎勵。

## 採礦速度提升 (MSB) 率乘數如何適用於不同類型的支持硬件？

硬件礦工可以註冊下列任何IOT硬件：
* 安全的 LPWAN 硬件
* 不安全的LPWAN硬件

安全的 LPWAN 硬件吸引承諾期為3、6、9、12、24或36個月的最高MSB乘數：1.025, 1.05, 1.075, 1.10, 1.15, 1.2

不安全的 LPWAN 硬件在同一承諾期內吸引下列較低的MSB乘數：1.0125、1.025、1.0375、1.075、1.1


## 鎖定硬件的承諾有多長時間？

3、6、9、12、24或36個月
