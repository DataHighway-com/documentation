以下角色是用户参与数据高速公路时使用的角色：

## 当前可用的角色

* Validator Node in Nominated Proof of Stake (NPoS)

## 未来的角色

* 在股权委托证明(NPoS) 中的排序器节点

* 在股权委托证明(NPoS) 中的提名器

* 数据购买者(在链际数据市场)

* LPWAN IoT 终端设备所有者(数据提供者)

* LPWAN IOT 网关所有者

* MXC 超级节点和其他 LoRaawAN 网络服务器所有者

* 网络所有者(LoRaWAN)

* 网络运营商所有者(LoRaWAN)

* 拥护矿工

* 开发矿工(链际数据市场应用开发者)

* （DHX DAO中的）治理矿工
